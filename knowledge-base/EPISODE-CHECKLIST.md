# 每集发布检查清单

## 内容

- [ ] 开场 10 秒内说明观众能学会什么
- [ ] 每个场景只有一个主要结论
- [ ] 专有名词有口语化解释
- [ ] 产品功能与官方资料一致
- [ ] 结尾包含验收动作和下一集预告

## 视觉

- [ ] 画布为 1920×1080，关键内容在安全区内
- [ ] 标题、截图、字幕无重叠和裁切
- [ ] 每个功能讲解页都有 WorkBuddy 界面截图和操作高亮
- [ ] 色彩明亮但正文对比度足够
- [ ] 每个场景有明确视觉焦点
- [ ] 转场不会遮挡字幕或制造闪烁

## 音频

- [ ] 女声清楚、语速自然
- [ ] 使用固定 `female_question_reference_999.wav`，哈希匹配且音色相似度 ≥ 0.60
- [ ] 保留真人教学式停顿、自然尾音和轻微句尾转折
- [ ] 场景长度覆盖完整旁白
- [ ] 没有削波、爆音或突然静音

## 工程与发布

- [ ] `npm test` 通过
- [ ] `npm run validate:content` 通过
- [ ] HyperFrames `npm run check` 通过
- [ ] MP4 完整解码且含音轨
- [ ] PPT 可打开并通过溢出检查
- [ ] 工程包、知识库和 GitHub 内容一致
